# Requirements

```bash
# Clone Repository
git clone https://github.com/Zevhys/personal-portfolio .

# Install Dependencies
npm install

# Run The Application
- npm run start (Run production app)
- npm run build (Compiles code)

# Running ESLint
- npx eslint .
```
